# Quick Start Guide - Live Market Data\n\nGet Stock Screener Pro running with live market data in 5 minutes.\n\n## Prerequisites\n\n- Node.js 18+ and npm\n- Internet connection\n- Free API key from one of: Finnhub, Alpha Vantage, Polygon, or IEX Cloud\n\n## Step 1: Get an API Key (2 minutes)\n\n### Quickest Option: Finnhub (Recommended)\n\n1. Visit https://finnhub.io\n2. Click \"Get Free API Key\"\n3. Enter email and click \"Get API Key\"\n4. Copy the API key (you'll see it immediately)\n\n**That's it!** Finnhub's free tier has:\n- 60 API calls/minute (plenty for 5000 stocks)\n- Real-time WebSocket support\n- No credit card required\n\n### Alternative: Alpha Vantage\n\n1. Visit https://www.alphavantage.co\n2. Scroll down to \"API key\"\n3. Enter your email\n4. Check email for API key\n\n## Step 2: Configure Environment (1 minute)\n\nCreate `.env.local` in the project root:\n\n```bash\n# For Finnhub\nFINNHUB_API_KEY=your_api_key_here\nNEXT_PUBLIC_DATA_MODE=live\n\n# For Alpha Vantage\n# ALPHA_VANTAGE_API_KEY=your_api_key_here\n# NEXT_PUBLIC_DATA_MODE=live\n```\n\n## Step 3: Install & Run (2 minutes)\n\n```bash\n# Install dependencies\nnpm install\n\n# Start dev server\nnpm run dev\n```\n\nOpen http://localhost:3000 in your browser.\n\n**Done!** The app is now using live market data.\n\n## Verify It's Working\n\n1. Check the HeroHeader - should show real stock data\n2. Open browser DevTools → Console\n3. Look for: `\"Loading * stocks in live mode...\"`\n4. Stock prices should match real market prices\n\n## Troubleshooting\n\n### Problem: \"Still showing simulated data\"\n\n1. Check `.env.local` exists in project root (not src/)\n2. Verify `NEXT_PUBLIC_DATA_MODE=live`\n3. Verify API key is correct\n4. Restart dev server: `Ctrl+C`, then `npm run dev`\n\n### Problem: \"API returns 401 Unauthorized\"\n\n- API key is incorrect or expired\n- Check for typos in `.env.local`\n- Generate new key from provider's dashboard\n\n### Problem: \"Rate limit exceeded\"\n\n- Finnhub free tier: 60 calls/min limit\n- Alpha Vantage free tier: 5 calls/min limit\n- Switch to hybrid mode: `NEXT_PUBLIC_DATA_MODE=hybrid`\n- Wait 1 minute before retrying\n\n## Mode Comparison\n\n| Mode | Data | Speed | API Key | Best For |\n|------|------|-------|---------|----------|\n| **simulated** | Generated | Instant | ❌ | Development, demos, offline |\n| **hybrid** | Top 20 real + rest simulated | Fast | ✅ | Balance of features & performance |\n| **live** | All real-time | Slower | ✅ | Production, real market analysis |\n\n## Next Steps\n\n1. **Explore real data** - Look at actual stock prices and data\n2. **Test with different modes** - Try hybrid or simulated\n3. **Deploy to production** - See [DEPLOYMENT.md](./DEPLOYMENT.md)\n4. **Customize filters** - Try searching for specific sectors\n5. **Watch portfolios** - Use the watchlist feature\n\n## Questions?\n\nRefer to [LIVE_DATA_INTEGRATION.md](./LIVE_DATA_INTEGRATION.md) for:\n- Detailed API provider comparison\n- Advanced configuration options\n- Rate limiting strategies\n- Deployment instructions\n- Troubleshooting guide\n\n## API Provider Features at a Glance\n\n### Finnhub ⭐ (Recommended)\n- Real-time WebSocket\n- 60 calls/min free\n- Company profiles\n- Technical indicators\n\n### Alpha Vantage\n- Free tier (5 calls/min)\n- Simple REST API\n- Technical indicators\n- Slower responses\n\n### Polygon.io\n- Enterprise-grade\n- End-of-day data free\n- Intraday requires paid\n- Excellent documentation\n\n### IEX Cloud\n- Reliable service\n- Free tier limited\n- Fast responses\n- Requires paid upgrade for most features\n
